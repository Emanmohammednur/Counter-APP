# Counter App

A modern responsive counter app built with HTML, CSS, Bootstrap, and JavaScript featuring sound effects, local storage, and glassmorphism UI.

## Features

- Increase, decrease, and reset counter
- Sound effects for interactions
- Sound toggle button
- Responsive design with Bootstrap
- Glassmorphism UI design
- Modern interactive buttons

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
